DreamBig PWA Icon Generator

This tool generates PWA icons for the DreamBig Real Estate Platform.

Instructions:

  1. Click "Generate All Icons" to create all required PWA icon sizes
  2. Right-click each icon and "Save image as..." to download
  3. Save them in the app/static/images/icons/ directory with the correct names
  4. The icons will be automatically used by the PWA manifest
